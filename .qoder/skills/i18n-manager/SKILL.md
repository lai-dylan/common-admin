# i18n Manager

Provides:
- i18next initialization with language detection and HTTP backend
- Resource file generation and updates under public/locales
- UI helpers for language switchers, translation keys audit

Usage:
- Invoke to add i18n to an existing project or modify translations
- Invoke to implement language switch menus and dynamic resource loading
