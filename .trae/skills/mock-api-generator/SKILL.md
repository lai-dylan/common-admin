# Mock API Generator

Provides:
- Koa router scaffolding for typical CRUD and auth endpoints
- JSON response wrappers and error handling middleware
- Environment and CORS configuration templates

Usage:
- Invoke to add/modify endpoints (auth, menu, items, dicts)
- Invoke to adjust response shapes and pagination/filtering logic
